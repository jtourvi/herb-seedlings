# herb-seedlings
Herbivory and nutrient status of experimental hardwood seedlings. Published in Ecosystems (2025)

# File and Data Description 
# Code (.R)

herb: Code for running data entry, cleaning, analysis, and data visualization

# Data (.csv)

VT_soil_analysis: Raw coil chemical values from UVM soil analytics lab.

coef_ac: Output of glmm environmental/biotic coefficiants for sugar maple seedling survival.

coef_fa: Output of glmm environmental/biotic coefficiants for American beech seedling survival.

coef_ac_bio: Output of glmm environmental/biotic coefficiants for sugar maple seedling biomass.

coef_fa_bio: Output of glmm environmental/biotic coefficiants for American beech seedling biomass.

corr_a: Output correlations values between soil and foliar nutrients at site-level for sugar maple.

corr_a_s: Output correlations values between soil/foliar nutrients and elevation for sugar maple.

corr_f: Output correlations values between soil and foliar nutrients at site-level for American beech.

corr_f_s: Output correlations values between soil/foliar nutrients and elevation for American beech.

foliar_n: Scaled foliar nutrients used for the purpose of model creation and data visualization. 

herb_b: Site-level percent browse for both seedling species.

herb_h: Site-level percent foliar herbivory for both seedling species.

soil_fig: Data used soley for manuscript figure creation. 

tissues_new: Raw foliar nutrients measured from Penn State Analytics lab.

trans_all: Raw seedling biometric data from 2 years of surveys.
